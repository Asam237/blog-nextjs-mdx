- Une base de données réactive
- Une facilité d’utilisation grâce à une API RESTful et une bibliothèque client JavaScript
- Des fonctionnalités d’authentification intégrées
- Un stockage de fichiers pratique
- Une évolutivité garantie par la gestion automatique de la mise à l’échelle de la base de données PostgreSQL
- Une intégration fluide avec Next.js via la bibliothèque client Supabase et des appels API à l’API RESTful de Supabase.
